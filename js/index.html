Tipos de dados

    Dados primitivos:

  • numbers, bigint
  • string
  • boolean
  • null e undefined

Variáveis

  • Declarando uma variável: Ex. var nome = "Lucas"
  • Declarando uma variável mutável: let
  • Declarando uma variável constante: const

Operações e Expressões

  • Operadores aritméticos: + | - | * | / | %
  • Operadores de atribuição: = += | -= | *= | /= | %=
  • Incremento e decremento: ++ | --
  • Agrupamento: (x + y) * n

Comentários

  • De uma linha: //
  • De várias linhas: /* ... */

Usando Javascript no HTML

  • Utilizando dentro do html a tag: string
  • Criando um arquivo .js e passar o caminho dentro da tag: script
